# Usage
## Camera
* `left click + drag`: rotate
* `wheel click + drag, alt + left click + drag`: move
* `scroll`: scale
* `right click`: reset

## Main Scene
* `Space`: switch axis display
* `Esc`: open editor

## Edit Mode
* `Esc`: close editor
* `Ctrl + K`: delete line
* `Ctrl + S`: Save
* `Ctrl + Shift + S`: Save As
* `Ctrl + L`: Load

The code takes effect immediately after you edit it.
